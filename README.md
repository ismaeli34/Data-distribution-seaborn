ğŸ“ˆ Key Visualizations Included
The notebook demonstrates several essential plots for analyzing data distributions:

Histplot / Distplot: Shows the frequency of data points within specific ranges (bins).

Jointplot: Compares the distribution of two different variables and shows the relationship between them.

Pairplot: Creates a matrix of plots to show distributions and relationships across every numerical column in a dataset.

Rugplot: Adds small vertical ticks along an axis to show exactly where every single data point is located.

ğŸ“‚ Dataset
The project uses the "Tips" dataset (built into Seaborn), which contains 244 records of restaurant data, including:

Total Bill: The cost of the meal.

Tip: The amount given to the server.

Size: The number of people in the dining party.
